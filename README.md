### Start coding in *Windows*
---
First install [Node.js](https://nodejs.org/en/download/) and [VSCode](https://code.visualstudio.com/download)
1. Install React in *CMD*
```
npx create-react-app react-app
cd react-app
code .
```
2. Run App in VSCode:
Open new Terminal
```
npm start
```
